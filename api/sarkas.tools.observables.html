  • dump_step (int) – Timestep interval saving.

  • k_list (list) – List of :math: k vectors.

Returns

  • vkt_perp_i (numpy.ndarray, complex) – Transverse velocity fluctuations along the \(x\) axis. Shape = ( no_species, no_dumps, no_ka_values)

sarkas.tools.observables.correlationfunction(At, Bt)[source]¶

Calculate the correlation function \(\mathbf{A}(t)\) and \(\mathbf{B}(t)\) using scipy.signal.correlate

\[C_{AB}(\tau) = \sum_j^D \sum_i^T A_j(t_i)B_j(t_i + \tau)\]

where \(D\) (= no_dim) is the number of dimensions and \(T\) (= no_steps) is the total length of the simulation.

Parameters
